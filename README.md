
# FocusTimer

A timer with dark mode option and four types of background noises to help you concentrate or just chill...

![FocusTimer Demo](./app-demo/focus-timer.gif)

https://stivenleite.github.io/focus_timer/


## Features

- 4 options of background noises: forest, rain, cafeteria and fire (it can be combined!)
- Light/dark mode toggle
- 5 minutes increments/decrements on timer
- Responsive layout
- Works in all devices
